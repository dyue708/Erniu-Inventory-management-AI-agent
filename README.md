# 库存管理系统

基于飞书表格的简单库存管理系统。

## 安装

1. 克隆项目
2. 创建虚拟环境：`python -m venv venv`
3. 激活虚拟环境：
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
4. 安装依赖：`pip install -r requirements.txt`
5. 配置 `.env` 文件 按照 `.env sample` 文件配置后 修改名称 为 `.env`

## 使用

1. 确保已配置好飞书应用和表格权限
2. 运行程序：`python src/main.py`
